# AGENTS.md

This is a personal GitHub homepage - a static markdown portfolio with profile badges, tech stack, and stats.

## Workflow

- **Every change must be committed** to git before completing the task
- No build or test commands needed - it's a static markdown site

## Structure

- `README.md` - Main profile page
- `images/` - Banner image and icons
- `resources/` - Reference files

## Making Changes

1. Edit relevant files
2. Commit with a descriptive message: `git add . && git commit -m "description"`
